# Projeto SOMPO — Predição de Risco de Sinistro Agrícola

## Objetivo
Desenvolver uma solução baseada em Machine Learning capaz de prever riscos de sinistro em equipamentos agrícolas.

## Tecnologias Utilizadas
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Metodologia
CRISP-DM

## Funcionalidades
- análise exploratória de dados (EDA)
- tratamento de variáveis
- análise de correlação
- identificação de fatores de risco
- preparação para modelagem preditiva

## Principais Insights
- operações próximas à água aumentam o risco
- chuva influencia diretamente os sinistros
- velocidades elevadas aumentam a exposição ao risco
- manutenção atrasada aumenta a probabilidade de incidentes

- geração de score de risco
- análise de importância das variáveis
- construção de dashboards
